How to Install Janus (Beta)

Janus is currently in beta and not yet available on the Chrome Web Store. You can still use it by installing as an unpacked extension in Chrome. Here's how:

  1. Download the Janus extension ZIP file from our site.
  2. Extract (unzip) the downloaded file to a folder on your computer.
  3. Open Chrome and go to chrome://extensions in your address bar.
  4. Enable Developer mode (toggle in the top right corner).
  5. Click Load unpacked and select the folder you extracted in step 2.
  6. Janus will now appear in your Chrome extensions. Pin it for easy access!
